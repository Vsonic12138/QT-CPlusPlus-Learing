## Jupyter-Book使用方式简单记录
常用指令：
```bash
jupyter-book build .
```
每次编辑完内容之后就执行该指令，重新生成html文件。
执行后，_build目录下会生成html文件，可以打开index.html文件查看。
根目录下的_toc.yml文件用于配置目录结构、_config.yml文件用于配置书籍信息。
